Built a books marketplace using django and react

Python, JavaScript
Django Rest Framework
React, Redux State Management
JWT Authentification
Axios
Postgres
  • Source code

This project is a book-selling platform that allows clients to search for and save books, create an account to manage their preferences, and communicate with sellers. Using a pre-existing dataset of books, sellers can manage their stores, create new book listings, and manage orders. Our platform provides a user-friendly interface for both buyers and sellers to connect and enjoy a seamless experience in buying and selling books.

    Features:
  • Browse and search books
  • Create reviews and rate books
  • Create and account and manage it
  • For Stores to Create based on an available books dataset
  • update and delete them
  • Manage the store
